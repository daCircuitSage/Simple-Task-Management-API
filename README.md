#  Simple Task Management API (Django REST Framework)

A simple and secure **Task Management API** built using **Django REST Framework**.  
This project allows users to register, log in, and manage their daily tasks — create, update, delete, and view — through clean and well-structured RESTful APIs.

---

##  Features

 User Registration & Login (Token Authentication)  
 Create, Read, Update, Delete (CRUD) tasks  
 Each user can only access their own tasks  
 API secured using DRF permissions  
 Easy to test using Postman or Talend API Tester  

---

##  Tech Stack

- **Python 3**
- **Django 5**
- **Django REST Framework**
- **SQLite** (default database)
